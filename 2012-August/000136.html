hi everyone!

a new version of my "EtherCAT XML Configuration Tool" is available at http://code.google.com/p/soem-configuration-tool/downloads/list

I've added the support for the cyclic frames with multi-command. So now is possible to manage the entire network with all the information stored in the ENI XML file.

in details:
1)I've added the parsing of the XML file to store the information about cyclic packets;
2)I've written the functions that prepare, send and receive the cyclic packets. these functions will replace ec_send_processdata and ec_receive_processdata.

These two function are still enable, so you can choose whether to configure the network via SOEM library or via XML file.


the wiki documentation is not updated, but the meaning of the new functions are straightforward.

I wrote this tool as an application layer for SOEM 1.2.5 no modifications are needed. I haven't tried yet if the tool works also with SOEM 1.2.8

any suggestions, critics and comments are welcolme!!

Rocco ^_^
