The device should work just out of the box. The most common problem is 
when the eeprom data in the device is not programmed correctly. In order 
to reprogram it, use beckhoffs twincat program, select the slave in 
offline mode, then connect online and update the eeprom with the xml 
data. Before you start copy the latest slave xml file from the beckhoff 
site in your twincat ethercat directory.

If this does not help, you could send me the wireshark capture of the 
configuration process, I'll try to figure out what is wrong with the 
sequence.

Arthur.
