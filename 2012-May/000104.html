Hi everyone!

finally I've completed the XML configuration tool, so you can configure your network from the XML file generated, for example, by TwinCAT.

the tool is made of two part: the parsing of the XML file and the State Machine implementation, in order to execute the state transitions according to the init commands' list contained in the XML file


I've made some test on my netX 500 and everythings worked fine, but I'm looking for some volunteers that could test this tool on other systems, maybe someone with  linux or eCos.

if interested, send me a mail and I'll give copy of the source code.

you can find the wiki of the project at: 

http://code.google.com/p/soem-configuration-tool/wiki/UserDocumentation
