> > I would like to start using SOEM but I have one question. Do I need 
> > an external Configuration tool (e.g. TwinCAT) to set up the network 
> > or is there a way to set the configuration via SOEM? Does SOEM read 
> > or generate ENI files?
> 
> 1. AFAIK, there are two ways to configure system with SOEM: (a)
>  using the information provided in configuration table (see 
> ethercatconfiglist.h for details) or (b) using online configuration 
> (SOEM reads necessary information from the EEPROM/Object Dictionary) 
> - see ethercatconfig.c/ethercatconfig.h for details. So, the answer 
> for your question is: you don't need external Configuration tool to 
> setup network in SOEM.
> 
> 2. No. There is no support for ENI/ESI read/generation in SOEM.
