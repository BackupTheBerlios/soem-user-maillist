Hi everyone!

in order to use less memory and in a more efficient way, I've changed the slave data array in a dynamic list, so a new element is add when a slave is found. in this way in memory there are only the information of the actual slaves, instead of a partial empty array of  EC_MAXSLAVE elements.

basically I replace 


ec_slavet    ec_slave[EC_MAXSLAVE];

with

ec_SlaveList *SlaveList;

where ec_SlaveList is:

typedef struct PACKED TagSlaveList
{ec_slavet ec_slave;
  struct TagSlaveList *nextSlave;
 } ec_SlaveList;

of course I had to modify ALL the functions that use ec_slave's fields.

I've made some tests and everithing seems to work fine. 

I can share the code if someone want it


Rocco ^_^

P.S.: in order to further memory's optimization I'm going to change also ec_eepromPDOt in a dynamic list
