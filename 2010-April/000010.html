As follow up:

SOEM before 1.2.4 did not support SoE slaves. The result of many emails 
is that SOEM now does support SoE.

Getting the AX5000 series to work is still not that simple because the 
drives do not store their configuration in the slave. The only way to 
get them in operational is to extract the info from the XML 
configuration file and feed that via the new SoE interface to the slave.
